# Hi! Welcome to TET-TET

**Try out the game [here](https://digital-cookie.github.io/TET-TET/).**

### `Discover`

A new playstyle of Tetris!

### `Compete`

For a new Highscore!

### `Enjoy`

Have fun playing!


## How to play

Use the 'arrow keys' on your keyboard to move and the 'space bar' to rotate the blocks.

## Twist

Unlike a normal Tetris game, the blocks in TET-TET fall in two different directions changing every block.

## More to come
